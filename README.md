# GitHub 搭建网站

> **注意项**
> * 项目起名推荐使用 `github_id.github.io`
> * 使用 `jekyll` 来创建初始目录

# 开始搭建

## 安装 `jekyll`
> 相关文档: [jekll 中文文档](https://www.jekyll.com.cn/docs/)

**安装完成 jekll 之后，创建一个全新的 jekll 网站，然后将生成的文件都复制到新建的 GitHub 项目下，这样就可以开始开发了。**
